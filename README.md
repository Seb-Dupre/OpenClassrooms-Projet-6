# Portfolio-architecte-sophie-bluel

Code du projet 6 d'intégrateur web.

## Information pour le lancer le code

 - Lancer le backend depuis votre terminal en suivant les instruction du fichier ReadMe.
 - Si vous désirez afficher le code du backend et du frontend, faites le dans 2 instances de VSCode différentes pour éviter tout problème


Deuxieme Projet JavaScript.

Ce projet nous fait utiliser pour la premiere fois un Kanban : https://openclassrooms.notion.site/da3bb5863a554b34ba1a8df90d4c99af?v=df7f8dcccd9f4917a664a559f00b7ccb&p=c10173024288498295c67b9625cf437f&pm=s
mais aussi l'utilisation de Swagger.

On y apprend donc la methode FETCH, et des requêtes d'API comme GET, POST, DELETE.
On apprend aussi créer une page de login et donc a gérer un état (connécté/non connécté) via la presence/absence d'un token qui est recupéré si l'identifiant et mot de passes fournis sont corrects.
